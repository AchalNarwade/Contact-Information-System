# 📱 Contact Information Display System

A Java-based Contact Management System built using **Swing (GUI)** and a **Console-based interface**, with a strong focus on **database integration using JDBC and MySQL**.

---

## 🚀 Features

- 📋 Add, Update, Delete Contacts
- 🔍 Search Contacts (Live search in UI)
- 💻 Console-based interaction
- 🖥️ Swing-based graphical interface
- 🗄️ MySQL database integration (JDBC)
- 📦 Structured backend using separate manager classes

---

## 🧠 Project Focus

This project primarily focuses on:

- Efficient **database operations (CRUD)**
- Clean separation between **UI and backend logic**
- Practical use of **JDBC for real-world applications**

> Note: UI has been kept simple intentionally, with more emphasis on backend design and database handling.

---

## 🛠️ Tech Stack

- Java (Core + Swing)
- MySQL
- JDBC

---

## 📂 Project Structure

- `Contact.java` → Model class
- `ContactManager.java` → Handles DB operations
- `DBConnection.java` → Database connection setup
- `ContactUI.java` → Swing-based UI
- `ConsoleApp.java` → Console-based interaction
